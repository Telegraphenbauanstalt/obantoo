OBanToo

Open Banking Tools

Projektseite bei Berlios Javadoc Webzugriff auf das CVS
Bei OBanToo handelt es sich um eine Bibliothek mit Tools für das Banking. Zur Zeit sind folgende Klassen realisiert:
  • DTAUS-Parser
    DTAUS-Dateien parsen für die Weiterverarbeitung. Momentan wird nur eine logische Datei pro physikalischer Datei unterstützt.
Geplant sind momentan weitere Klassen:
  • DTAUS-Writer
    DTAUS-Dateien ausgeben.
  • Kontonummer-Checker
    Java-Implementierung der Bibliothek ktoblzcheck
    • OBanToo benötigt JDK 1.4 und steht unter LGPL.
