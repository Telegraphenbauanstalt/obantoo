OBanToo

Open Banking Tools

 Autor: Heiner Jostkleigrewe
Projektseite bei Berlios  |  Javadoc |  Webzugriff auf das CVS |  Download
Bei OBanToo handelt es sich um eine Bibliothek mit Tools für das Banking. Zur Zeit sind folgende Klassen realisiert:
  • DTAUS-Parser
    DTAUS-Dateien parsen für die Weiterverarbeitung. Seit Version 0.3 werden auch mehrere logische Dateien pro physikalischer Datei unterstützt.
  • DTAUS-Writer
    DTAUS-Dateien mit einer oder mehreren logischen Dateien erstellen. Pro logischer Datei können entweder Überweisungen oder Lastschriften eines Kontos enthalten sein. Ein Mixed-Inhalt ist nicht zulässig.
  • QIF-Parser
    QIF-Dateien parsen.
  • QIF-Writer
    QIF-Dateien erstellen

OBanToo wird künftig innerhalb von Hibiscus eingesetzt.

OBanToo benötigt JDK 1.4 und steht unter LGPL.
